# 配置 GOPATH

`GOPATH` 是一个环境变量，用来表明你写的 `go` 项目的存放路径

`GOPATH` 路径最好只设置一个，所有的项目代码都放到 `GOPATH` 的 `src` 目录下。



![GO目录结构](https://pic-1257412153.cos.ap-nanjing.myqcloud.com/images/2022/12/12/7-51e238.png)

### windows

打开系统环境变量，添加新项 `GOPATH` 到你写代码的地方
