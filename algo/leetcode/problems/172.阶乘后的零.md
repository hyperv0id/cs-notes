# 阶乘后的零
#算法/数学
[172.阶乘后的零](https://leetcode-cn.com/problems/factorial-trailing-zeroes/)
## 题目描述
给定一个整数 $n$ ，返回 $n!$ 结果中尾随零的数量。
提示 $n! = n * (n - 1) * (n - 2) * ... * 3 * 2 * 1$

示例 1：
```
输入：n = 3
输出：0
解释：3! = 6 ，不含尾随 0
```

示例 2：
```
输入：n = 5
输出：1
解释：5! = 120 ，有一个尾随 0
```

示例 3：
```
输入：n = 0
输出：0
```

提示：`0 <= n <= 104`

进阶：你可以设计并实现对数时间复杂度的算法来解决此问题吗？


## Solution
$\because 10 = 2\times 5$
所有只需要计算2和5哪个少
5肯定比2少，就算 5 的个数

```c++
class Solution {
public:
	int trailingZeroes(int n) {
		int no5 = 0;
		int i=5;
		while(i<=n){
			no5 += n/i;
			i*=5;
		}
		return no5;
	}
};
```
- **时间复杂度**：$O(log n)$
- **空间复杂度**：$O(1)$
