## 题目描述

给定一个长度为 $N$ 的数列，求数值严格单调递增的子序列的长度最长是多少。

#### 输入格式

第一行包含整数 $N$。

第二行包含 $N$ 个整数，表示完整序列。

#### 输出格式

输出一个整数，表示最大长度。

#### 数据范围

- $1 \leq N \leq 1000$
- $−10^9$≤数列中的数$\leq 10^9$

#### 输入样例：

```
7
3 1 2 1 8 5 6
```

#### 输出样例：

```
4
```

## 题目分析

1. 如果数组长度只有1，那么就是1
2. 如果数组长度为2，判断第二个是否大于第一个
	- 如果大于，返回2
	- 小于等于，返回1
3. 那么对于数组长度为2的情况下，
	1. 对于数组中每一个元素$i$，先假设**以自己结尾的**最大上升子序列为1
	2. 然后，从左到右依次**向前看**，更新以自己结尾的最大上升子序列
	3. 如何更新：$dp_i = \max(dp_i, dp_j+1)$ 其中 $j<i$

## 代码实现

```C++
#include<bits/stdc++.h>
using namespace std;

int main(){
	// 初始化
    int n;cin>>n;
    vector<int> arr(n);
    for(int i=0;i<n;i++){
        cin>>arr[i];
    }
    
    // 先假设最长子序列为1
    vector<int> dp(n, 1);
    
    for(int i=0;i<n;i++){
	    // 向前看
        for(int j=0;j<i;j++){
            if(arr[i]>arr[j]){
	            // 更新
                dp[i] = max(dp[i], dp[j]+1);
            }
        }
    }
    cout << dp.back()<<endl;
    return 0;
}
```
